# Adaptive Learning System - Frontend

Next.js frontend for the Adaptive Learning System POC.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env.local` file:
```
NEXT_PUBLIC_API_URL=http://localhost:8000
```

3. Run the development server:
```bash
npm run dev
```

The app will be available at http://localhost:3000

## Features

- Google OAuth authentication
- Teacher dashboard (create flows, add questions, view progress)
- Student dashboard (browse flows, take adaptive quizzes, track progress)
